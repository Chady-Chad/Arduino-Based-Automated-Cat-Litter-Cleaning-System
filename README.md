## 🐱 Automated Cat Litter System

This project is an **Automated Cat Litter System** that I developed in collaboration with a group of college Electrical Engineering students during my freelance work in Arduino and electronics fabrication.

I was responsible for assisting in the **design, programming, and implementation** of the system. The goal of the project was to create machine that automates the cleaning process of a cat litter box, improving convenience, hygiene, and efficiency for pet owners.

The system integrates **microcontroller-based control (Arduino)** with sensors and actuators to detect usage and trigger automated cleaning mechanisms. It was designed with a focus on reliability, safety, and ease of use.

This project highlights my hands-on experience in:
- Arduino programming  
- Electronics design and prototyping  
- Automation and control systems  

It also reflects my ability to collaborate with engineering teams and deliver functional, real-world solutions.
